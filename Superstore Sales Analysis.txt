# 🛒 Superstore Sales Analysis

## 📌 Project Overview
Analysis of Superstore sales data using Python and Matplotlib.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Charts Created
- Sales by Category
- Profit by Category
- Monthly Sales Trend
- Ship Mode Distribution

## 📁 Dataset
Sample Superstore dataset with 10,000+ rows of sales data.

## 🔍 Key Insights
- Technology is the highest revenue category
- Consumer segment contributes the most sales
- Standard Class is the most used shipping mode
